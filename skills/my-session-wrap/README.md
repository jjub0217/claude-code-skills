# my-session-wrap

세션 종료 시 작업 정리, 문서 업데이트, 학습 기록을 자동으로 수행하는 스킬.

## 기능
- Git 상태 기반 세션 변경사항 자동 요약
- 4개 병렬 에이전트: 문서 업데이트, 자동화 제안, 학습 정리, 후속 작업 제안
- 중복 검증 에이전트로 결과 정리
- 사용자 선택에 따라 커밋, 문서 업데이트, 자동화 생성 실행

## 설정
별도 설정 없이 사용 가능합니다. 현재 프로젝트의 git 상태를 기반으로 동작합니다.

## 사용법
```
/my-session-wrap
```

## 예시
세션 마무리 시 "마무리해줘"라고 요청하면, 변경사항을 분석하고 문서 업데이트/자동화/학습 정리를 제안한 뒤 선택한 작업만 실행합니다.
