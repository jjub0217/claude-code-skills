# study-roadmap

채용공고와 이력서를 비교 분석하여 갭 분석 → 맞춤 프로젝트 제안 → 공부 로드맵을 자동 생성하는 스킬.

## 기능
- 채용공고 자격요건/우대사항 vs 이력서 경험 갭 분석
- 갭을 채울 수 있는 맞춤 프로젝트 제안
- 압축형/여유형 두 가지 로드맵 생성
- 면접 대비 핵심 질문 예상

## 설정
| 구분 | 항목 | 기본값 |
|------|------|--------|
| 선택 | 로드맵 출력 경로 | `~/Desktop/` |

## 사용법
```
/study-roadmap
```

## 예시
채용공고 URL과 이력서를 제공하면, 자격요건 대비 현재 수준을 분석하고 맞춤 프로젝트와 주간 공부 로드맵을 생성합니다.
