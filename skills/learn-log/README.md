# learn-log

세션 히스토리를 분석하여 Q&A, 인사이트, 의사결정을 주간 학습 로그에 자동 기록하는 스킬.

## 기능
- 세션 히스토리(`.jsonl`)를 자동 탐색하여 학습 내용 추출
- 4개 병렬 에이전트: Q&A 추출, 인사이트 추출, 산출물 추출, 블로그 후보 감지
- 주간 학습 로그 마크다운 파일에 자동 append (중복 방지)

## 설정
| 구분 | 항목 | 기본값 |
|------|------|--------|
| 선택 | 히스토리 파일 경로 | `~/.claude/projects/` 하위 |
| 선택 | 학습 로그 파일 경로 | 프로젝트별 설정 |

## 사용법
```
/learn-log
```

## 예시
세션 종료 후 "학습 정리해줘"라고 요청하면, 오늘 대화에서 나온 Q&A와 인사이트를 자동으로 주간 학습 로그에 정리합니다.
