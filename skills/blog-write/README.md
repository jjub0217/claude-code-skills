# blog-write

학습 로그의 블로그 후보를 기반으로 기술 블로그 초안을 자동 작성하는 스킬.

## 기능
- 학습 로그에서 블로그 후보 주제를 자동 파싱하여 목록 제시
- 3단계 에이전트: 자료 수집 → 아웃라인 작성 → 초안 작성
- 1인칭 경험 기반 서술로 자연스러운 블로그 초안 생성

## 설정
| 구분 | 항목 | 기본값 |
|------|------|--------|
| 선택 | 학습 로그 파일 경로 | `~/Desktop/weekly-learning-log.md` |
| 선택 | 블로그 출력 경로 | `~/Desktop/blog-drafts/` |

## 사용법
```
/blog-write
```

## 예시
"블로그 써줘"라고 요청하면, 학습 로그에서 블로그 후보 목록을 보여주고 선택한 주제로 기술 블로그 초안을 작성합니다.
