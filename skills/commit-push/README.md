# commit-push

커밋 → 푸시 → PR 생성을 자동화하는 스킬.

## 기능
- 변경사항 자동 분석 및 커밋 메시지 생성
- 브랜치명에서 이슈 번호 자동 추출
- PR 템플릿에 맞춘 PR body 자동 작성
- 이슈 자동 close 키워드 포함

## 설정
| 구분 | 항목 | 기본값 |
|------|------|--------|
| 필수 | GitHub 레포 | - |
| 선택 | base 브랜치 | `develop` |
| 선택 | PR body 템플릿 (chore) | 프로젝트에 맞게 설정 |
| 선택 | PR body 템플릿 (fix/feat) | 프로젝트에 맞게 설정 |
| 선택 | Co-Author | 없음 |

## 사용법
```
/commit-push
```

## 예시
작업 완료 후 "커밋하고 PR 만들어줘"라고 요청하면, 변경사항을 분석하여 커밋 메시지를 제안하고 확인 후 푸시 및 PR을 생성합니다.
