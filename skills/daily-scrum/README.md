# daily-scrum

GitHub 커밋 기반으로 Daily Scrum 페이지를 자동 생성하는 스킬.

## 기능
- 어제/오늘 git 커밋 내역 자동 분석
- 전일 업무 + 금일 예정 업무 자동 작성
- Notion DB에 페이지 생성 또는 로컬 마크다운 생성

## 설정
| 구분 | 항목 | Notion 필요 |
|------|------|-------------|
| 필수 | GitHub 레포 | 불필요 |
| 필수 | base 브랜치 | 불필요 |
| 선택 | DAILY SCRUM DB | 필요 |
| 선택 | 사용자 ID / 이름 | 필요 |

Notion 미설정 시 `scrum/` 폴더에 마크다운으로 생성됩니다.

## 사용법
```
/daily-scrum
```

## 예시
"데일리 스크럼 만들어줘"라고 요청하면, git 커밋 이력을 분석하여 오늘의 Daily Scrum 페이지를 자동 생성합니다.
